Отчёт может быть сформирован на период не более, чем за: 90 (календарных дней). Начало периода отчёта не может быть позже, чем за: 90 (календарных дней)

Отчет об операциях по счету формируется за период не более 90 календарных дней. Дата начала периода может быть выбрана не ранее, чем за 90 календарных дней до текущей даты.

Включение проведенных операций в отчет, а также отражение остатка по счету на начало выбранного периода осуществляется с учетом отражения операций в системе банка.

Получить выписку за любой период обо всех операциях, отраженных по счету, можно обратившись в любое учреждение банка при наличии документа, удостоверяющего личность.
